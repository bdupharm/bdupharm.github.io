Debugging Javascript in IE8

May 5, 2016

First time debugging javascript in IE8 today, here is what I wish I knew going in:

When you select a DOM node/element in IE8 you get a useless representation of the object.

If you want to look at any of the methods or values on the object you have to type in the name of the variable into the watch tab of the developer tools.
